# Homelab 2: DHCP Failure & Troubleshooting (SOC-Focused)

## Tool Used
- Cisco Packet Tracer

## Objective
To analyze DHCP behavior and troubleshoot IP assignment issues
commonly encountered in enterprise environments.

## Lab Components
- Router
- Switch
- End Devices (PCs)
- DHCP-enabled network

## Configuration Performed
- Enabled DHCP for automatic IP assignment
- Configured router interfaces
- Connected devices through switch

## Issue Encountered
- DHCP failure
- End device received APIPA address (169.254.x.x)

## Troubleshooting Steps
- Verified physical and logical connectivity
- Checked DHCP pool configuration
- Validated router interface status
- Corrected configuration to restore DHCP functionality

## SOC Relevance
- Identified abnormal network behavior
- Practiced structured incident triage
- Learned how misconfigurations trigger alerts
- Improved SOC Level 1 troubleshooting mindset
